# Строки

Популярные методы:

```jsx
	'string'.length  
	// длина строки
	
  for (let char of "string") 
  // перебор символов строки
  
  "string".charAt(index) 
  // возвращает указанный символ из строки. index от 0 до -1
  
  "string".concat('str1', str2, str3) 
  // объединяет текст из двух или более строк и возвращает новую строку.
  
  "string".endsWith('ab')
  // заканчивается ли строка символами указанными в скобках,возвращая,
  // соответственно, true или false.
 
  "string".startsWith('ab') 
  // начинается ли строка символами указанными в скобках,
  // возвращая, соответственно, true или false.
 
  "string".indexOf('Widget', pos) 
  // ищет подстроку substr в строке str, начиная с позиции pos,
  // и возвращает позицию, на которой располагается совпадение, либо -1.
  
  "string".lastIndexOf(substr, position) 
  // начинает искать с конца
  "string".includes(substr, pos) 
  //  возвращает true, если в строке str есть подстрока substr, 
  // либо false, если нет.

  "string".slice(start [end]) 
  // Возвращает часть строки от start до end (не включая). 
  // Если аргумент end отсутствует, slice возвращает символы до конца строки.
  
  "string".substring(start [end]) 
  // Возвращает часть строки между start и end (не включая) end.
  
  "string".substr(start [length]) 
  // Возвращает часть строки от start длины length.
  
  "string".trim() 
  // удаляет пробельные символы с начала и конца строки

  "string".split(разделитель, limit длина эл arr) 
  // возвращает массив, разбивает объект String на массив строк
  //  путём разделения строки указанной подстрокой.
  
  "string".repeat(count)
   //  возвращает новую строку, содержащую указанное количество
   // соединённых вместе копий строки, на которой он был вызван.
   
  "string".replace(regexp|substr, newSubStr) 
  //  возвращает новую строку с некоторыми или всеми сопоставлениями
  //  с шаблоном, заменёнными на заменитель.
  
  "string".replaceAll(pattern, replacement) 
  // возвращает новую строку со всеми совпадениями pattern,
  // который меняется на replacement.
  
  "string".search([regexp]) 
  // При успехе метод search() возвращает индекс первого сопоставления с
  //  регулярным выражением внутри строки. В противном случае метод вернёт -1.
```
